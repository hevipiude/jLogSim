Tietoja ohjelmasta

 

 

Ohjelman avulla voidaan suunnitella ja testata digitaalipiirejä, eli käyttäjä voi piirtää haluamansa piirin ja tämän jälkeen ajaa piirin logiikkaa.

Ohjelma toimii Windows/Linux-ympäristössä, kunhan koneelle on asennettu Java tuki. Alla olevissa linkeissä on helpot Java tuen asentamiseen tarvittavat ohjeet ja tiedostot.

 

Windows-ympäristöön:                             http://www.java.com/en/download/windows_automatic.jsp 

Linux ympäristöön:                                    http://java.com/en/download/help/linux_install.jsp#enable        

tai sivuilta:                                                  http://www.java.sun.com
